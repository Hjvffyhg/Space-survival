# Space Survival

**Concept**

The project is a **top-down 2D space shooter** game where the player takes on the role of a lone fighter pilot defending their sector against the relentless Kla'ed armada across evolving Kardashev Civilization eras. The game focuses on **bullet-hell combat, resource management, and survival mechanics**, where players must collect credits, unlock stronger vessels, and survive infinite scaling waves. Each stage becomes progressively more challenging, requiring both timing and strategic enemy priority management based on CPU scheduling algorithms.

**Inspiration**

The game is inspired by classic and modern space shooters for its engaging bullet-hell combat and wave-based progression. It also draws mechanical and progression inspiration from snake games like *Worm Zone* and classic *Snake*, where players must continuously move, survive, and grow in power in chaotic environments. Additionally, it integrates unique ideas from computer science operating system task schedulers (F CFS, RR, and HRRN), mapping abstract logic directly onto battlefield targeting.

**Color Palette**

A carefully selected palette is used to create mood, readability, and immersion:

- **Deep Space / Midnight Blue (#0A0F1F)** – Used for the dark vacuum of space and neutral UI backgrounds.
- **Neon Cyan (#00D9FF)** – Represents the player ship, friendly projectiles, and primary glowing UI elements.
- **Laser Red / Orange (#EF4444)** – Highlights enemies, aggressive projectiles, warning systems, and damage text.
- **Electric Indigo (#6366F1)** – Used for secondary UI contrast, shop elements, and special upgrades.

**Assets**

The project will use a combination of assets sourced from [Void Fleet Pack 1 by foozlecc](https://foozlecc.itch.io/void-fleet-pack-1) along with assets generated by nano banana and ChatGPT:

- **Player Character Sprites** – Idle and engine-thrusting animations for various ship classes (Fighter, Scout, Bomber, etc.).
- **Enemy Sprites** – Different behaviors such as grunts, tanks, drones, and massive multi-phase capital bosses.
- **Background Layers** – Starfields, nebulas, and deep space environments.
- **Collectibles** – Credits, health packs, shield recharges, and power-ups for progression.
- **UI Elements** – Health bars, score counters, pause menus, hangar shop, and dynamic radar visuals.
- **Audio Assets** – Background music, laser blast sounds, explosive impacts, and ambient UI feedback.

**Hardware Requirements:**

- Minimum: Intel i3 / AMD equivalent, 4GB RAM, integrated graphics
- Recommended: Intel i5 or higher, 8GB RAM, dedicated GPU

**Software Requirements:**

- Game Engine: React, TypeScript, HTML5 Canvas API
- Graphics Editor: Photoshop, Canva, or Krita (for custom sprites)
- Audio Editor: Audacity or FL Studio (basic use)
- Code Editor: Visual Studio Code

**Development Requirements:**

- Organized project structure (components, libs, assets)
- Version control system (Git/GitHub recommended)
- Testing environment for debugging and balancing gameplay (Internal AI Tester Agent built-in)

**Controls**

The game uses simple and responsive controls for smooth gameplay experience:

- **W / A / S / D Keys** – Move up, left, down, and right
- **Mouse Click** – Fire main weapons and navigate menus
- **Mouse Movement** – Aim weapons towards the crosshair
- **Shift Key** – Engage Boost / Engine Overdrive
- **Q Key** – Dash / evasive maneuver
- **P Key** – Toggle AI Tester Auto-Pilot
- **Esc Key** – Pause game and open settings menu

Controls are designed to be **intuitive, responsive, and beginner-friendly**, while still supporting advanced gameplay mechanics.
